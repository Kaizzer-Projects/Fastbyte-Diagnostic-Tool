# ⚡ FastByte Diagnostic Tool (Pre-Alpha)

Ferramenta de diagnóstico de hardware em tempo real desenvolvida em Python com PySide6.

O FastByte exibe informações detalhadas do sistema de forma visual e simples, incluindo CPU, RAM e Disco em tempo real.

---

## 📊 Funcionalidades atuais (Pre-Alpha)

- 🖥️ Informações do sistema
- 🧠 Monitoramento da CPU (uso e frequência em MHz)
- 🧩 Monitoramento de RAM (uso, frequência e slots)
- 💾 Monitoramento de Disco (uso e saúde)
- 📈 Barras de uso em tempo real
- 📄 Geração de relatório em PDF
- 🧭 Interface gráfica em PySide6

---

## ⚙️ Tecnologias utilizadas

- Python 3.13+
- PySide6 (GUI)
- psutil (monitoramento de sistema)
- reportlab (geração de PDF)
